# Mera Ashaina

📱 **Overview**
Mera Ashaina is a real estate app that connects property seekers with agents. Users can browse properties, search, and contact sellers directly. Agents can create property listings, while admins manage approvals.

🎯 **Problem it Solves**
Simplifies property buying, selling, and rental processes by providing a digital platform for users and agents.

✨ **Features**
- Splash screen
- User & agent login/register
- Home & profile screens
- Property search & filter
- Agents can create property listings
- Admin can approve/reject listings and view status
- Users can view property details and make calls to sellers/buyers
- Dark mode & other UI features

🛠 **Tech Stack**
- Flutter
- Firebase Authentication
- Firestore / Realtime Database
- State Management (Provider / Bloc optional)

🧠 **Architecture**
- MVVM / Clean Architecture
- High-level separation of UI, business logic, and data layers
